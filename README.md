# PriceWise API 🚀

Smart product price tracking and decision engine.

## Features

- Real Amazon India scraping
- Smart product matching
- Historical price tracking
- JWT authentication
- Protected routes
- User accounts
- Wishlist system
- Price alerts
- AI buy/wait decisions
- Rate limiting
- Caching
- Modular backend architecture

---

# Tech Stack

- Node.js
- Express.js
- Supabase
- JWT
- Cheerio
- Axios
- Render
